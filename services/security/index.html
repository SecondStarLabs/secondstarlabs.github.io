Secure web and mobile application development

Security is Key

In a world of email hacks and million-record data breaches, application security is a top concern for users and organization leaders.

We stay at the front of new developments in security and compliance. We invest our time in learning the latest tools and guidelines, and we track new vulnerabilities that may affect our clients.

Programming secure web and mobile apps

We use mature web application frameworks such as Ruby on Rails and Elixir Phoenix that contain defaults to help avoid common vulnerabilities involving code injection, session hijacking, and cross-site scripting.

Cloud security and compliance

We deploy to Heroku , which is built atop Amazon Web Services. Heroku and Amazon manage their platform updates for network security, vulnerability testing, and anti-virus checks. They keep their platforms compliant with the ISO 27001 standard and maintain IT governance certifications such as FedRAMP and SOC 2.

Shared responsibility model

Open source software and cloud platforms are a head start but must be used by an experienced team who have documented security procedures.
